# Sudoku task
Игровое поле представляет собой квадрат размером от 9х9 и более, массово распространены примеры до 25х25, разделённый на меньшие квадраты со стороной в 3 и более клеткок, т.е. всё поле – размером от 81 клетки до 625 клеток. В них уже есть некоторые числа (по количеству клеток в меньшем квадрате) – «подсказки» (можно загружать из текстового файла или из файла xml или опционально реализовать GUI). Требуется заполнить свободные клетки числами от 1 до 25 так, чтобы в каждой строке, в каждом столбце и в каждом малом квадрате 5х5 каждое число встречалось бы лишь один раз. Алгоритм требуется подобрать самостоятельно, параллельная версия должна использовать технологию OpenMP и директивы task.
