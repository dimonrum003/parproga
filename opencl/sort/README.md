# Sort task
Требуется создать программу с использованием языка программирования C или C++ и API OpenCL, которая обеспечила бы сортировку массива данных с использованием вычислений на GPGPU. Подобрать наиболее подходящий для данных условий алгоритм сортировки. Реализовать программу Обеспечить проверку корректной работы программы. Произвести измерение времени работы программы для массивов разного размера.
